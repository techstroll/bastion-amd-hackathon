# 🛡️ Bastion — Private AI Cloud-in-a-Box on AMD

**Showcase instance** for the AMD Developer Hackathon: ACT II (Unicorn Track).

This Space runs the real Bastion router, policy classifier, metrics, and
tamper-evident audit code with a **simulated engine** (the GPU deployment on
AMD Developer Cloud is shown in the submission video). Traffic is auto-seeded
so the Console is alive when you open it.

- **Repo:** https://github.com/techstroll/bastion-amd-hackathon
- **What to try:** Overview (live routing + egress counter) · Departments
  (onboard a new department live) · Compliance (export the hash-chained audit,
  verify the chain) · GPU (telemetry)

One AMD GPU. Every department. Zero egress.
